# 项目简介

本人有记录学习笔记的习惯，采用文件夹和`.md`(Tppora+Obsidian)文件进行记录，而图片使用 cdn+本地静态服务器搭建。这是一个无感的博客项目，仅仅将本来的笔记文件夹关联到项目 docs 就能生成 一个博客网站。
以前用过 csdn、博客园、Hero 等博客一键项目，都没法好好的跟自己的笔记方式进行结合，丢失好多原创文章，这次选择这种文件夹+md 文件的方式，希望可以确保以后不会在丢失笔记。

- 2021-09-20 项目使用 `vuepress` 创建
- 2023-09-20 项目迁移至 `Docusaurus-v2`
- 2025-02-01 项目升级 `Docusaurus-v3`

## 项目缘由

本身本人就有记录学习笔记的习惯，很久很久以前用的是 CSDN，随着 CSDN 的资本化和网络上各种书各种音的低质量复制粘贴，任何有点作用的原创内容或者学习记录都会被各种机械人，各种无底线的同行进行自我收录和自我原创。

所以我觉得身为一个以后还打算会战根多年的码农，自己的东西还是需要好好保管好。

## 项目架构

首先本项目的核心是一个文件夹，里面所有的笔记或者学习资料均以 markdown 文件记录，是一个纯粹的文件夹多层结构。

以这种结构通过市面上多种博客框架或者文档框架进行搭建，最后选一个自己喜欢的
